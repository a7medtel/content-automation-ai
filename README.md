# AI Content Automation 🤖

AI-powered social media content generator built with n8n, Groq AI, and Telegram.

## What it does
- Receives topic and platform via Telegram
- Generates professional post using Groq AI (Llama 3.1)
- Saves content to Google Sheets Content Calendar
- Sends generated content back via Telegram

## Tech Stack
- n8n (workflow automation)
- Groq AI / Llama 3.1 (content generation)
- Google Sheets (content calendar)
- Telegram Bot (interface)
- Railway (hosting)
